# Faker::Superhero

Available since version 1.6.2.

```ruby
# Random Superhero name
Faker::TvShows::Superhero.name #=> "Magnificent Shatterstar"

# Random Superhero power
Faker::TvShows::Superhero.power #=> "Photokinesis"

# Random Superhero prefix
Faker::TvShows::Superhero.prefix #=> "Captain"

# Random Superhero suffix
Faker::TvShows::Superhero.suffix #=> "the Fated"

# Random Superhero descriptor
Faker::TvShows::Superhero.descriptor #=> "Bizarro"
```
